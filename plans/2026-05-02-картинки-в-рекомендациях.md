#### API grsai.ai

- **Base URL:** `https://grsaiapi.com` (есть китайский зеркальный хост `https://grsai.dakka.com.cn`).
- **Submit:** `POST /v1/draw/completions`
  - Headers: `Authorization: Bearer <GRSAI_API_KEY>`, `Content-Type: application/json`.
  - Body: `{"model": "gpt-image-2", "prompt": "...", "aspectRatio": "1:1", "quality": "high", "webHook": "-1", "shutProgress": true}`.
  - Ответ синхронный — `{code:0, msg:"success", data:{id:"<task-id>"}}`. (Если `webHook != "-1"`, было бы стрим/колбэк — нам не подходит для batch.)
- **Poll:** `POST /v1/draw/result` с `{"id":"<task-id>"}` каждые ~3 секунды до `data.status in ("succeeded", "failed")`.
- **Результат:** `data.results[0].url` — публичный URL PNG/JPEG, живёт **2 часа** → качаем сразу.
- **Ошибки:** `failure_reason ∈ {"input_moderation", "output_moderation"}` → не ретраим, логируем и пропускаем пункт; `error == "error"` → 1 ретрай (рекомендация в доке).
